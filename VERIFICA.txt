LÍNEA DE ACCIÓN
════════════════════════════════════════════════════════════════════════════════

AHORA (0 minutos):
  1. Abre una terminal
  2. cd TestOnline
  3. npm install (ya está hecho)
  4. npm run dev

INMEDIATO (1 minuto):
  5. Abre http://localhost:3000 en navegador
  6. Escribe nombre y sala ID
  7. Haz clic "Unirse a Sala"

PARA MULTIJUGADOR (2 minutos):
  8. Abre otra pestaña con http://localhost:3000
  9. Usa el mismo nombre y sala ID
  10. ¡Ahora están conectados!

PARA JUGAR (3+ minutos):
  11. WASD para mover
  12. Mouse para apuntar
  13. Click para disparar
  14. ¡Diviértete! 🎮

COMANDOS PRINCIPALES
════════════════════════════════════════════════════════════════════════════════

Instalar dependencias:
  $ npm install

Ejecutar en desarrollo:
  $ npm run dev

Parar el servidor:
  Ctrl+C en la terminal

Verificar puerto:
  $ lsof -i :3000

Desplegar en Vercel:
  $ npm install -g vercel
  $ vercel --prod
