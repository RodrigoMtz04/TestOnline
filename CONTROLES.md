# CONTROLES Y GUÍA DEL JUEGO

## 🎮 Controles Básicos

### Movimiento
| Tecla | Acción |
|-------|--------|
| **W** | Arriba |
| **A** | Izquierda |
| **S** | Abajo |
| **D** | Derecha |
| **Flechas** | Movimiento alternativo |

### Combate
| Acción | Controles |
|--------|-----------|
| **Apuntar** | Mueve el mouse |
| **Disparar** | Click izquierdo |
| **Mirar alrededor** | Mueve el mouse |

## 📊 Sistemas de Juego

### Sistema de Salas
- Cada sala tiene una ID única (ej: `room_a1b2c3d4`)
- Máximo 8-10 jugadores por sala (recomendado)
- Las salas se eliminan cuando todos los jugadores salen

### Sistema de Vida
- Vida inicial: 100 HP
- Daño por disparo: 10 HP
- Mueres cuando llegas a 0 HP

### Sistema de Puntuación (Futuro)
- +10 puntos por enemigo eliminado
- +5 puntos por daño causado
- Ranking de salas

## 🎯 Estrategias

### Principiante
1. Practica el movimiento y apuntería
2. Aprende el tamaño del mapa
3. Juega defensivamente

### Intermedio
1. Domina el movimiento diagonal
2. Usa el espacial del mapa a tu favor
3. Anticipa movimientos de enemigos

### Avanzado
1. Circle strafe (movimiento circular mientras disparas)
2. Predicción de disparos
3. Control del área de juego

## 🐛 Solución de Problemas

### No puedo conectar con otros jugadores
- Verifica que ambos usen la misma sala ID
- Recarga la página
- Consulta la consola (F12)

### Los movimientos se ven entrecortados
- Es normal en conexiones lentas
- Mejora tu conexión a internet
- Reduce procesos de fondo

### El juego se congela
- Cierra otras pestañas
- Reinicia el navegador
- Limpia el cache del navegador

## 💡 Consejos

- **Practica primero en una sala privada** con amigos
- **Customiza tu nombre** para identificarte fácilmente
- **Usa el chat** si está disponible para comunicarte
- **Respeta el fair play** - no hacks ni trucos
